# Advanced Data Analytics Project

## Project Overview

This project focuses on advanced SQL analytics techniques built on top of a SQL Server Data Warehouse.

The goal is to transform raw business data into meaningful insights through analytical SQL queries, customer segmentation, product performance analysis, KPI reporting, and business intelligence reporting.

---

## Objectives

- Explore business data across dimensions and measures
- Perform trend and time-series analysis
- Analyze product performance
- Conduct part-to-whole contribution analysis
- Perform customer segmentation
- Build customer reports
- Build product reports
- Create reusable SQL reporting views
